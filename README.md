# Como rodar o projeto

Não há necessidade de nenhum pacote externo ou coisa do tipo, é necessário somente o node.

É esperado que você inicie o backend e depois inicie o front end. Para isso, faça o seguinte:

```
cd backend
npm install
npm run dev
```

**OBS: O backende roda na porta 3000 em localhost, então é necessário ter esta porta aberta.**
Após estes três comandos, o backend estra funcionando.

Agora para o frontend temos

(Voltando para o readme)
```
cd ..
```

Iniciando o frontend:
```
cd backend
npm install
npx vite
```

Com isso, tanto o front end como o backend estão rodando.
